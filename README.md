# URL Shortener with great A/B testing feature

using NodeJs, Express, and MongoDB

# Installation

 - `git clone ...`
 - `mongo`
    - `use url_shortener`
    - `db.counters.insert({ _id: 'url_count', seq: 1 })`
 - `node app.js`
